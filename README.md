# mmfashion

## Introduction

mmfashion is an open source object detection toolbox based on PyTorch.


## Installation

1. Install from pypi.

    ```bash
    pip install mmfashion
    ```

2. Install from source.

    ```bash
    git clone https://github.com/open-mmlab/mmfashion.git
    python setup.py install
    ```
